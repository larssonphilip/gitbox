Gitbox

Elegant everyday tool for human beings.

Gitbox is a graphical interface to Git version control system. In a single window you see the branch history and file status.

Everyday operations are easy: stage and unstage changes with a checkbox. Pull, merge and push with a single click. Double-click the change to show the diff with FileMerge.app

Other features

Automatic. Gitbox updates status each time you focus its window. When window is out of focus, it periodically fetches data from the selected remote branch and displays unmerged commits in the history.

Convenient. History shows all commits on both local and remote branch. Non-pushed commits are green. Non-pulled commits are grey.
